## Installation
- Install the following dependencies
```bash
pnpm add @radix-ui/react-progress flubber
```

- Copy and paste the following code into your project
<ComponentSource title='src/components/ui/progress.tsx'>
    <Progress />
</ComponentSource>

## Usage
```tsx
import { Progress } from '@/components/ui/progress'
```

```tsx
<Progress value={30} />
```
