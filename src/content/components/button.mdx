## Installation
- Install the following dependencies
```bash
pnpm add @radix-ui/react-slot
```

- Copy and paste the following code into your project
<ComponentSource title='src/components/ui/button.tsx'>
    <Button />
</ComponentSource>

## Usage
```tsx
import { Button } from "@/components/ui/button"
```

```tsx
<Button>Button</Button>
```

## API Reference

### Button

| Prop | type | default |
| :--- | :--- | :--- |
| variant | `default` \| `tonal` \| `outline` \| `elevated` \| `text` | `default` |
| size | `sm` \| `md` \| `lg` | `md` |
| square | `boolean` | `false` |
| icon | `boolean` | `false` |
| asChild | `boolean` | `false` |
