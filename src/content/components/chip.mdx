## Installation
- Install the following dependencies
```bash
pnpm add @radix-ui/react-slot
```

- Copy and paste the following code into your project
<ComponentSource title='src/components/ui/chip.tsx'>
    <Chip />

## Usage
```tsx
import { Chip } from '@/components/ui/chip'
```

```tsx
<Chip>Chip</Chip>
```
